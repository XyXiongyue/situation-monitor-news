GIT Coatings在Posidonia 2026上与劳埃德注册公司一起推出XGIT-VORTEK ™，下一代石墨烯基螺旋桨涂层

2026-06-05 01:08

国际航运新闻04/06/2026

GIT Coatings在Posidonia 2026期间与劳合社（Lloyd ' s Register）在LR x GIT Coatings客户类型认证仪式上正式推出了其下一代石墨烯基螺旋桨涂层系统XGIT-VORTEK ™。

XGIT-VORTEK ™是GIT Coatings螺旋桨涂层技术的下一次发展，旨在帮助船东超越螺旋桨抛光作为定期恢复措施，转向长期性能保持。该系统的设计旨在解决当前方法的主要局限性：抛光事件之间的污垢堆积、气穴相关的涂层损坏以及干船坞周期中螺旋桨光滑度的损失。

XGIT-VORTEK ™由GIT的两亲性石墨烯纳米网络（AGN）技术提供支持，是一种专用的三层螺旋桨涂层系统。防垢面涂层有助于防止清洁螺旋桨随着时间的推移而附着污垢，而增强的中间涂层充当系统的结构层，有助于吸收气穴应力并减少叶片边缘和尖端的涂层损坏。总而言之，该系统旨在支持更强的涂层完整性、长期的光滑度保持以及在干船坞周期中更稳定的螺旋桨性能。

关键性能特征包括：·最多可节省5%的燃油·阻止污垢附着，帮助螺旋桨随着时间的推移保持清洁·增强中涂层有助于抵抗气穴应力，以随着时间的推移更强的涂层完整性·低摩擦表面旨在帮助维持螺旋桨在干船坞周期中的光滑性·提供性能保证

随着船东越来越关注生命周期效率和排放性能，推进优化正成为运营战略的关键组成部分。螺旋桨效率仍然是降低商业船队燃料消耗、排放和维护成本的最直接的操作杠杆之一。

螺旋桨状况直接影响燃料消耗、CII性能、EU ETS暴露和总体运营成本。借助XGIT-VORTEX™，船舶运营商可以超越反应抛光，并通过专为干船坞周期设计的更耐用、更可靠的螺旋桨涂层系统来支持更稳定的螺旋桨性能。

此次活动由GIT Coatings首席执行官Mo AlGermozi和Lloyd ' s Register首席增长官Andy McKeran出席，汇集了来自全球海事部门的船东、运营商、行业利益相关者和客户。

GIT Coatings首席执行官Mo AlGermozi在Posidonia 2026获得劳氏船级社首席增长官Andy McKeran的型式认证证书，突出了可持续推进器性能技术的下一个发展

Mo AlGermozi表示：“XGIT-VORTEK ™反映了我们为全球海事行业提供可持续、高性能涂层技术的下一步使命。”“随着船东继续优先考虑效率、减排和运营灵活性，该行业正在越来越多地转向围绕长期性能、可持续性和可衡量运营价值设计的技术。”

该仪式在Posidonia 2026年期间在劳埃德纪事报展位举行，标志着船舶性能技术发展的又一个里程碑，并凸显了行业朝着可持续、高性能涂层系统的发展势头，旨在支持运营效率和脱碳目标。

GIT Coatings首席执行官Mo AlGermozi在Posidonia 2026年劳埃德纪事报首席增长官Andy McKeran以及咨询服务主管Philippos Sfiris、Thanos Kostakopoulos GIT Coatings区域业务发展总监和Margos SA董事总经理Yiannis Makris获得了型号批准证书

此次发布是在劳合社（Lloyd ' s Register）最近对GIT Coatings的下一代石墨烯基船体涂层XGIT-FORCE™进行了类型批准之后进行的，进一步加强了LR和GIT Coatings之间围绕先进船舶涂层技术创新和技术保证的合作。

安迪·麦克伦评论道：“海事部门继续寻求能够支持运营效率和可持续发展目标的创新技术。很高兴加入Posidonia的GIT Coatings，该公司介绍了其在石墨烯基船舶涂层技术方面的最新进展。”

自2022年以来，GIT Coatings在全球已超过约600种船舶和螺旋桨涂层应用，为油轮、散货船和更广泛的商业航运领域的船东和运营商提供支持。

参加Posidonia 2026的参观者有机会与GIT Coatings和Lloyd ' s Register的代表会面，讨论积极主动的船体和螺旋桨性能管理策略以及石墨烯基涂层技术在支持海上脱碳目标方面的作用。来源：石墨创新技术公司
